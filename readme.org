*** description
a 2D Eulerian fluid simulation using a staggered grid, implemented using 1D MVectors. Explained parameters are at the beginning of [[Main.hs]]. Dependencies in [[./default.nix]]
*** notes
- simulation will become unstable if parameters (overrelaxation, dt, ..) arent balanced
- requires zlib, and optionally imagemagick/ffmpeg for converting frames -> video
- can be measured in frames per second rather than seconds per frame for a 192x108x50 simulation
