*** description
a 2D Eulerian fluid simulation using a staggered grid, implemented using 1D MVectors. Explained parameters are at the beginning of [[Main.hs]]
*** notes
- parameters at top of Main.hs, with notes
- simulation will become unstable if parameters (overrelaxation, dt, ..) arent balanced
- requires zlib, and optionally imagemagick/ffmpeg for converting frames -> video (see [[./default.nix][default.nix]])
- can be measured in frames per second rather than seconds per frame for a 192x108x40 simulation
